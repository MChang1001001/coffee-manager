# Coffee Manager

咖啡豆管理软件，目标是记录咖啡豆入库信息、评价、冲煮方案与冲煮复盘。

## Project Structure

```text
backend/   Spring Boot 后端
frontend/  Vue 3 前端
docs/      产品、数据库、接口设计文档
```

## Local Requirements

- JDK 17+
- Maven 3.9+
- Node.js 20+
- MySQL 8+

## Next Steps

1. 配置 `backend/src/main/resources/application.yml` 中的数据库账号密码。
2. 在 `frontend` 目录安装依赖并启动开发服务。
3. 根据 `docs` 下的设计文档继续补充实体、接口和页面。
